# Specs Rules

## Format

- Terms: Capitalize first letter (e.g., "a list of Records").  
- Attributes: **bold**.  
- Actions: _italic_.  
- Otherwise: use lowercase; avoid bold/italic unless necessary.  

Wrong: 
```markdown
**Attributes:**
- **Path**: Location of the record.
```
Correct:
```markdown
Attributes:
- **path**: location of the Record.
```
